Project Overview

Purpose

The Election Information System is a database-driven application designed to modernize election data management by supporting electronic voting and advanced result analysis. It provides a secure platform for voters to cast their votes and offers tools for analyzing Bundestag election results from 2017 and 2021.

Key Objectives
  • Enable secure electronic voting at polling stations.
  • Analyze and compare results of Bundestag elections (2017 & 2021).
  • Improve transparency and facilitate deeper insights into voting trends.

Features

  • e-Stimme: Secure electronic voting system with fraud prevention mechanisms.
  • Votealyze: Intuitive interface for result comparison with advanced analytics.

  • Interactive graphs and filtering for detailed result exploration.

  • Backend: Python for secure vote processing and data management.
  • Frontend: React for a responsive, user-friendly interface.
